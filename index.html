About The Game

Cryptocurrency Donations: a Technological Method of Charity


Cryptocurrencies are becoming an increasingly popular form of donation. Fidelity Charitable noted that cryptocurrency are the fastest growing category of assets donated, growing 140% faster than real estate, stock shares, and other illiquid assets. Cryptocurrency donations bring a number of benefits to the sector. It is faster, cheaper, and more transparent than fiat currency donation. This project aims to capitalize on the trend of cryptocurrency donation by creating an in-browser crypto miner as a new form of charity donation. Several charities have implemented group cryptocurrency pools to raise funds such as UNICEF. However, it has not been widely explored in the form of in-browser miners, especially after the discontinuation of CoinHive. The purpose of this website is to test user engagement and determine if in-browser mining coudl be a feasible method of charity donation in the future.

About this Project and Website


The Problem

In-browser mining is incredibly inefficient. Most crypto mining are done on machines with graphic cards. To be profitable, there must be a large number of individuals mining for a long period of time. Therefore, there must be a large amount of user engagement.

Secondly, most in-browser miners used CoinHive to mine Monero. CoinHive became discontinued in 2018, resulting in a large number of these websites and miners to become obsolete.


The Interdiciplinary Solution: Game Elements

The solution is to increase engagement to keep users on the miner for a long period of time. In terms of in-browser methods to improve user engagement, games have often been implemented to entertain users. The longer you are on the website playing a game, the more your computer mines and the more donations is given to the charity. For this purpose, a simple game that largely replicates the Google Chrome dinosaur game is presented to the user with a few gaming elements that increases engagement.


The Project:

This project is to test the feasibility of using a game to improve the feasibility of in-browser mining for charity purposes. The website does not currently have a miner, and is more used as a test to see user engagement and determine mining profitability.

The Cryptocurrency

In the future, the website will use CoinImp, a service that uses JavaScript for Browser Mining. It uses visitor CPU resources when you are on the website in replacement of advertisements.


What happens when you click 'The Game'

This website is simply examining how users will interact with the game and if it functions to keep people on it for a long enough time. There are currently no mining scripts that will run on your browser.
